## 6. `/etc/hosts` na tomto Macu

**Stav:** záznam **chýba** (`MISSING`).  
**Sudo / osascript:** vyžaduje **lokálne admin heslo** v GUI dialógu — agent ho nemôže zadať bez vás.

Ak sa dialóg „osascript wants to make changes“ ešte ukázal → zadajte Mac heslo.  
Inak v Termináli:

```bash
sudo sh -c 'grep -q cms.growmedica.cz /etc/hosts || echo "37.9.175.131 cms.growmedica.cz" >> /etc/hosts'
sudo dscacheutil -flushcache
sudo killall -HUP mDNSResponder
open -na "Google Chrome" --args --incognito "https://cms.growmedica.cz/"
```

Po plnej DNS propagácii (aj 8.8.8.8 + bez Tailscale problémov) riadok z `/etc/hosts` odstráňte.
